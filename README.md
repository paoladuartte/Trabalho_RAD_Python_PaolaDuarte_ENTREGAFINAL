📘 Contextualização do Projeto

Este projeto consiste no desenvolvimento de um sistema simples de gerenciamento acadêmico, baseado nas operações de CRUD (Create, Read, Update e Delete), voltado para uma ESCOLA PREPARATÓRIA que oferece cursos básicos de diferentes áreas.

A proposta do sistema é permitir o controle de alunos, cursos e matrículas, possibilitando o cadastro, consulta, atualização e exclusão dessas informações de forma estruturada. Esse tipo de aplicação simula um cenário real encontrado em instituições de ensino de pequeno porte, que necessitam de uma solução prática e funcional para organizar seus dados.
A modelagem do sistema foi construída com base em três entidades principais: alunos, cursos e matrículas, sendo esta última responsável por representar o relacionamento entre alunos e os cursos nos quais estão inscritos.

Para o armazenamento dos dados, foi utilizado o SQLite, um sistema de gerenciamento de banco de dados leve, de fácil configuração e que não requer servidor. A escolha do SQLite se deu pelo fato de o projeto ser de pequeno porte, onde não há necessidade de uma infraestrutura mais robusta, tornando-o ideal para aplicações locais e de menor complexidade.
